Two-Way Colorful Chat Messenger (Java Swing)

A clean and enhanced two-way chat simulation desktop application built using Java Swing.
Both users can chat with each other using separate panels, colorful backgrounds, and real-time synced messages.
This project demonstrates UI design, event handling, custom painting, and Swing layout management.
